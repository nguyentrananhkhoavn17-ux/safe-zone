# SAFE ZONE - Web (scaffold)

This folder contains a minimal scaffold for the SAFE ZONE web PWA and a small Express API stub.

Quick start:

1. Install dependencies

```bash
cd web
npm install
```

2. Run the API stub

```bash
npm run start:server
```

3. In another terminal run the frontend dev server

```bash
npm run dev
```

Open http://localhost:3000 and the app will fetch hazards from http://localhost:4000/api/v1/hazards
